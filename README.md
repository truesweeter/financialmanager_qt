## Менеджер личных финансов

Приложение для учёта личных финансов.  
Приложение позволяет пользователю создать учётную запись, авторизоваться и вести список своих финансовых транзакций.  
Все данные сохраняются в базе SQLite и доступны при последующих запусках.

## Возможности программы

- Регистрация нового пользователя  
- Авторизация существующего пользователя  
- Добавление транзакций двух типов: **доход** и **расход**  
- Ввод суммы, даты и описания транзакции  
- Отображение всех транзакций текущего пользователя в таблице  
- Проверка корректности вводимых данных (сумма, дата, выбор типа операции)


## Назначение

Программа предназначена для простого ведения личных финансов и хранения информации о доходах и расходах в локальной базе данных.
